Behavioral Neuroscience & Systems Modeling

With either physical or VR mazes, navigational paradigms give us a good window for various aspects of brain function.
Sensory information no longer present to the subject may be needed again for the current task. Working memory maintenance is a key component of flexible animal behavior.
Circuits might engage in activity patterns not related to the animal's current sensory state, where the pattern can represent past experiences or future action plannings.
Using reinforcement learning agents trained on the same tasks as animals, systems neuroscience can probe how task demands shape behavior, representations, and computations in the brain.

Neural Decoding & Machine Learning

Feature-gated RNN trajectory decoder maps from a trajectory of PVs to experiment variables including animal spatial location, travel direction, and experiment phase.
Modeling location decoding as a classification task presents challenges to the model to understand the relationship between input and output.
Mathematical representation of a specialized loss function used to train my decoder model. This "GICEL" loss function improves model generalization by "encoding spatial geometry of the output dimensions".

Reinforcement Learning & Computational Modeling

In reinforcement learning, PO-MDP (Partially Observable Markov Decision Process) provides a mathematical framework for modeling working memory tasks such as the DNMTS (Delayed Non-match to Sample) task.
The amazing TMazing API models a DNMTS (Delayed Non-match to Sample) T-maze navigation task with PO-MDP (Partially Observable Markov Decision Process) framework. This implementation extends the OpenAI Gym interface, a standardized toolkit for custom RL environments.
The TMazing console (implemented in Pygame) provides a graphical interface for interacting with the TMazing environment. The lower-right panel displays simulated place cell activity traces generated using the Gaussian bump model.

Connectomics & Optical Neuroimaging

STN (subthalamic nucleus) is the main modulator of basal ganglia output, making it an important component of the highly-recurrent basal ganglia circuit. Employing a sophisticated viral tracing strategy called TRIO, we map the major contributors to STN activity.
The TRIO viral tracing strategy uses a clever combination of retrograde tracers with the Cre expression system to restrict the expression of fluorescence reporter to a specific path of projection.
Multiphoton imaging provides a highly precise and minimally invasive methodology for probing neural dynamics in deep brain tissue. We demonstrate that three-photon microscopy can be used to image the CNS structures of an un-manipulated adult invertebrate with fine anatomical detail.
