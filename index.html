Shown below are some relevant topics and works related to my research.

With physical and VR maze setups, navigational paradigms give us a good window for various aspects of brain function.
Sensory information no longer present to the subject may be needed again for current task, which needs to be maintained as working memory.
Circuits might engage in activity patterns not related to the animal's current sensory state, where the pattern can represent past experience or future action planning.
Using reinforcement learning agents trained on the same tasks as animals, systems neuroscience can probe how task demands shape behavior, representations, and computations in the brain.
Attention-based RNN decoder architecture mapping from a trajectory of PVs to experiment variables including animal spatial location, travel direction, and experiment phase.
Transforming spatial location representation provides challenges to the model to understand the relationship between input and output.
Mathematical representation of a specialized loss function used in dynamic loss function scheduling. The "GICEL" loss function improves model generalization and inform the model of the spatial geometry of the output dimensions.
In reinforcement learning, a Markov Decision Process (MDP) provides a mathematical framework for modeling how an agent learns to choose actions based on states and rewards.
The amazing Tmazing API was designed to model a typical DNMTS (Delayed Non-match to Sample) T-maze navigation task. This implementation extends the OpenAI Gym Env interface with using a PO-MDP (partially observable MDP) framework.
The Tmazing console provides a graphic interface using a lightweight game library for interacting with the Tmazing API. It can also be used for rendering when running/training RL agents on the task.
TRIO analysis, a sophisticated viral-based tracing method, allows us to map the neural connectivity of the subthalamic nucleus.
Analysis at the injection site confirms fluorescent signals from rabies and TVA: the implementation of the construct is successful.
Three-photon microscopy allows us to image the CNS structure of an un-manipulated adult invertebrate with fine anatomical detail.
